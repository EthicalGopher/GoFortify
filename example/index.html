GoFortify is a reverse proxy firewall that inspects incoming requests for malicious patterns and blocks them before they reach your backend server.

Features

  • SQL Injection Protection: Detects and blocks SQL injection attacks by analyzing query parameters, form bodies, and JSON payloads for suspicious SQL patterns.
  • XSS Protection: Blocks cross-site scripting (XSS) attempts by detecting malicious script tags and JavaScript event handlers.
  • Rate Limiting: Limits requests per IP address to prevent denial-of-service (DoS) attacks and brute force attempts.
  • CORS Support: Configurable cross-origin resource sharing headers.

How It Works

  1. Client sends a request to GoFortify (this proxy)
  2. GoFortify intercepts the request and inspects:
    • Query string parameters (GET requests)
    • Form data (application/x-www-form-urlencoded)
    • Multipart form data
    • JSON bodies (application/json)
  3. If malicious patterns are detected, the request is blocked with a 403 Forbidden response
  4. Valid requests are forwarded to your backend server
